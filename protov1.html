Key Features

FlowAI provides a suite of powerful features designed to augment your development process seamlessly.

Real-time Memory Capture

File watcher, process monitor, and clipboard listener build a timeline of your work.

Vector-based Recall

Embeds and indexes your activity, allowing for semantic search over your entire history.

Auto-generated Summaries

Daily and weekly digests of your progress, ready for stand-ups and reports.

Context-aware Prompting

Automatically injects relevant code snippets and history into your LLM prompts.

Obsidian Integration

Syncs your development memory with your knowledge base for deeper insights.

Lightweight Desktop App

The Flow State Cycle

FlowAI is designed to support the natural, cyclical process of creative development. Hover over a step to learn more.

1. Code

Write code, experiment, and build. FlowAI silently observes file changes and terminal commands.

2. Test

Run tests, check outputs, and validate your work. Errors and successes are logged as part of the context.

3. Reflect

Query your recent activity. Ask "Why did that test fail?" or "Show me the API I was using."

4. Document

Generate a commit message, a changelog entry, or a knowledge base article from the captured flow.

AI Personas

FlowAI uses specialized agents, or "personas," to handle different aspects of the workflow, ensuring expert assistance for every task.

Context Agent

The core agent that watches your files and actions, building the foundational memory of your work.

Prompt Agent

Enriches your queries to other LLMs with relevant context from your memory, improving response quality.

Doc Agent

Analyzes your activity timeline to draft documentation, commit messages, and knowledge base articles.
